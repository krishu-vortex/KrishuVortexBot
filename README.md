# KRISHU-VORTEX WP BOT 🚀

Telegram-based WhatsApp Multi-Device Bot with 500+ commands

## Features
- 🔢 Pairing Code (no QR)
- 📱 Works with any number (+91, +92, all countries)
- 🤖 500+ Commands
- 🎥 Media Download (144p - 1440p)
- 🧠 AI Chat (Gemini/Meta)
- 🆓 24/7 Free on Render
- 📱 Mobile-friendly setup

## Quick Deploy
1. Fork this repo
2. Connect to Render.com
3. Add your Telegram Bot Token
4. Bot is LIVE!

## Commands
- /start - Welcome
- /connect <number> - Pair WhatsApp
- /menu - All commands
- /download <url> - Download media
- /ai <text> - AI Chat
